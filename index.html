HTML TAGS CHART

S.N Tag Description
1 <!DOCTYPE> Defines the document type
2 <html> Defines an HTML document
3 <head> Defines information about the document
4 <title> Defines a title for the document
5 <body> Defines the document's body
6 <h1> to h6> Defines HTML headings
7 <p> Defines a paragraph
8 <br< Inserts a single line break
9 <hr> Defines a thematic change in the content
10 <!--...--> Defines a comment
11 <abbr> Defines an abbreviation or an acronym
12 <address> Defines contact information for the author/owner of a document/article
13 <b> Defines bold text
14 <bdi> Isolates a part of text that might be formatted in a different direction from other text outside it
15 <>
16 <bdo> Isolates a part of text that might be formatted in a different direction from other text outside it
17 <blockquote> Defines a section that is quoted from another source
18 <cite> Defines the title of a work
19 <code> Defines a piece of computer code
20 <del> Defines text that has been deleted from a document
21 <dfn> Represents the defining instance of a term
22 <i> Defines a part of text in an alternate voice or mood
23 <ins> Defines a text that has been inserted into a document
24 <kbd> Defines keyboard input
25 <mark> Defines marked/highlighted text
26 <meter> Defines a scalar measurement within a known range (a gauge)
27 <pre> Defines preformatted text
28 <progress> Represents the progress of a task
29 <q> Defines a short quotation
30 <rp>
31 <script> Defines a client-side script
32 <datart> Links the given content with a machine-readable translation
33 <ruby> Defines a ruby annotation (for East Asian typography)
34 <s> Defines text that is no longer correct
35 <samp> Defines sample output from a computer program
36 <small> Defines smaller text
37 <sub> Defines subscripted text
38 <sup> Defines superscripted text
39 <u> Defines text that should be stylistically different from normal text
40 <var> Defines a variable
41 <wbr> Defines a possible line-break
42 <img> Defines an image
43 <map> Defines a client-side image-map
44 <area> Defines an area inside an image-map
45 <audio> Defines sound content
46 <ul> Defines an unordered list
47 <ol> Defines an ordered list
48 <li> Defines a list item
49 <dl> Defines a description list
50 <dt> Defines a term/name in a description list
